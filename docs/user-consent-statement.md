# User Consent Statement Template

This prototype is created for ICT111 learning purposes. We will ask users to test the prototype and provide feedback. The information collected will be used only to improve the project prototype. Users should not provide sensitive personal information. Participation is voluntary.

## Data Collected
- Name/nickname or anonymous ID: Anonymous tester ID only (e.g. S01, A01) — no real name is collected.
- Feedback score: Ease-of-use and usefulness ratings (1–5).
- Comments about usability: Free-text comments about the experience, task by task.
- Task completion result: Whether each assigned task (e.g. search the schedule, submit a report, update a class status) was completed successfully.

## Data Use
The data will be used for class project analysis, prototype improvement, and final project reporting. It will not be shared outside the ICT111 course, sold, or used for any purpose beyond this class project.
